# Page Archetype System

Each page type has its own deterministic layout model.

## /About or /Mission Page

**Algorithm**:

1. Small hero
2. Vision block (centered text)
3. Three (3) pillars
4. Timeline or programs
5. CTA band
   _Never improvise_.

## /Event Page

**Algo**

1. Hero (dark)
2. Event essentials grid (3 cards)
3. What to expect (2 column)
4. Gallery preview
5. CTA

## /Pricing Page

**Algo**

1. Hero
2. Three (3) pricing cards
3. Feature comparison table
4. FAQ
5. CTA
6. /Checkout
7. 2-column layout
8. left: summary
9. right: payment form
10. trust badges
11. security reassurance
12. Never centered layout.

## /Dashboard page

**Algo**

1. Rule set:
2. sidebar fixed
3. content max-w-none
4. grid stats row
5. data table
6. activity feed
7. Different constraints than marketing pages.

## /Admin page

**Algo**

1. left nav
2. content area
3. filter/search row
4. table
5. modals
6. Functional layout > aesthetic rhythm.
